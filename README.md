### ozon-tracker

## Возможности
- Периодическая проверка статуса (раз в час).
- Уведомления об изменении статуса.
- Ручная проверка статуса по команде.

## Зависимости
aiohttp
aiogram

## Как получить Cookie из браузера

Cookie нужны, чтобы Ozon не блокировал запросы бота. Они живут от нескольких дней до месяца, после чего их нужно обновить.

1.  Откройте в браузере (Chrome, Firefox) страницу отслеживания вашей посылки.
2.  Нажмите **F12**, чтобы открыть Инструменты разработчика.
3.  Перейдите во вкладку **"Сеть" (Network)**.
4.  Обновите страницу (**F5**).
5.  В списке запросов найдите тот, который начинается с `tracking/...`.
6.  Справа найдите раздел **"Заголовки запроса" (Request Headers)**.
7.  Найдите строку `Cookie:` и скопируйте **всю длинную строку** после двоеточия.
8.  Вставьте эту строку в файл `.env` в переменную `YOUR_COOKIES`.
